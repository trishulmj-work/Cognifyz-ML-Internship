# Cognifyz Technologies – Machine Learning Internship

This repository contains my completed tasks for the **Machine Learning Internship** at [Cognifyz Technologies](https://www.cognifyz.com), focused on predictive modeling, recommendation systems, and data analysis using a restaurant dataset.

## 📌 Tasks Completed

| Task | Description | Notebook |
|------|-------------|----------|
| Task 1 | Predict Restaurant Ratings – regression model to predict a restaurant's aggregate rating based on its features | `TASK-1.ipynb` |
| Task 2 | Restaurant Recommendation – content-based filtering system to recommend restaurants based on user preferences | `TASK-2.ipynb` |
| Task 3 | Cuisine Classification – classification model to predict a restaurant's cuisine type | `TASK-3.ipynb` |
| Task 4 | Location-based Analysis – geographical analysis of restaurant distribution, ratings, and pricing by city/locality | `TASK-4.ipynb` |

## 🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Folium (for geographical visualization)

## 📂 Dataset

The dataset used (`Dataset .csv`) contains restaurant details including cuisine, location, price range, and ratings, provided as part of the internship program.

## 📊 Approach

Each notebook follows a consistent workflow:
1. Data preprocessing (handling missing values, encoding categorical variables)
2. Model building (regression/classification/recommendation logic)
3. Evaluation using appropriate metrics
4. Insight extraction and interpretation
